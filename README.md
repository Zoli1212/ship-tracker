Ship Tracker App
Welcome to the Ship Tracker App! This application allows you to track and explore information about different ships.

Getting Started
To get started with the Ship Tracker App, follow these steps:

Install dependencies using npm:
npm install
Start the development server:
npm start


Used Technologies
The Ship Tracker App is built using the following technologies:

Redux: State management library.
Redux Toolkit: Official toolset for efficient Redux development.
RTK Query: Part of Redux Toolkit, used for data fetching and caching.
TailwindCSS: Utility-first CSS framework for styling.
React Testing Library: Testing utilities for React components.
Functionalities
The Ship Tracker App offers the following functionalities:

List All Ships: The app displays a list of all available ships, showing their basic information.

Async Searching by Owners: The app allows you to search for ships by their owners asynchronously. The search feature updates in real-time as you type.

Ship Details: Clicking on a particular ship in the list will take you to a details page where you can view comprehensive information about that ship.

Testing
The Ship Tracker App's functionalities are thoroughly tested using the React Testing Library. These tests ensure that the app behaves correctly and maintains its functionality as changes are made.
